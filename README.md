# Icons
The icons here are based on [semantic ui icons](https://semantic-ui.com/elements/icon.html)

# Usage

```html
<html>
    <head>
        <link rel="stylesheet" href="https://struckchure.github.io/icons/css/icons.css">
    </head>

    <body>
        <span>
            <i class="icon user"></i>
        </span>
    </body>
</html>
```
